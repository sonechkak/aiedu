# Проект анализа данных

Кратко: набор скриптов для предобработки и визуализации датасета (pandas, seaborn, matplotlib). Проект разрабатывается в `PyCharm` на macOS внутри виртуального окружения `venv`/`.venv`.

## Требования
- Python 3.10+ (в проекте используется Python 3.12)
- Установленные пакеты (пример): `pandas`, `numpy`, `seaborn`, `matplotlib`
- Рекомендуется использовать виртуальное окружение

## Установка
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

## Структура проекта
- `data/` - папка с исходными данными
- `notebooks/` - Jupyter ноутбуки для анализа и визуализации
- `README.md` - документация проекта
- `requirements.txt` - список зависимостей
- `LICENSE` - лицензия проекта

## Использование
1. Активируйте виртуальное окружение:
```bash
source .venv/bin/activate
```

2. Запустите Jupyter Notebook для анализа данных:
```bash
jupyter notebook
```
